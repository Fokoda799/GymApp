# 🏋️‍♂️ GymFuel

**GymFuel** is a complete AI-powered fitness ecosystem built to help users track nutrition, workouts, coaching, and progress in real time.

It includes:
- 📱 Mobile app (Flutter)
- ⚙️ Backend API (Express.js + Socket.IO)
- 🖥️ Admin dashboard (Next.js)
- 🎙️ Real-time voice calls (WebRTC)
- 🤖 AI fitness assistant
- 🧠 Smart coaching system

## 🚀 Features

### 📊 Fitness Tracking
- Daily calorie tracking (food intake + burned calories)
- Weight and progress statistics
- Personalized fitness goals

### 🍽️ Nutrition System
- Create custom recipes
- Add food entries
- AI-assisted nutrition recommendations

### 🏋️ Training System
- Full exercise library with explanations & tutorials
- Equipment guide
- Personalized workout programs from coaches

### 🧑‍🏫 Coaching System
- Coach invitation system
- Coach-client program management
- Real-time chat (Socket.IO)
- Voice calls (WebRTC)

### 🤖 AI Assistant
- Smart chatbot aware of user profile
- Nutrition + workout guidance
- App usage assistant

### 🔐 Authentication System
- Secure login/register
- Password recovery
- Role-based system (Client / Coach / Admin)
